# 作业要求：编写基于手机的室内定位传感器数据采集程序(或使用已有APP)，提取相关数据集进行简要分析，提交一份数据分析报告

# 题目：使用手机陀螺仪和磁力计检测姿态的对比实验分析
## 实验内容

我们使用phyphox APP来进行采集，尝试解答以下问题：
用phyphox中陀螺仪数据做积分和用磁力计的数值都可以判断手机姿态方向，但这二者有何区别？精度如何？

## 误差特性
陀螺仪积分磁力计主要误差零偏漂移 + 噪声积分累积噪声 + 磁场畸变短时准抖长时漂移随 t 线性增长稳定（无累积漂移）环境敏感否极敏感（金属、电脑、磁铁干扰严重）
这也是为什么实际系统几乎都用互补滤波 / 卡尔曼滤波把两者融合,你这个实验的最终结论很可能就是"两者各有所长所以要融合"。

## 实验设计
phyphox 可以导出 CSV,建议用"Sensors"实验同时记录陀螺仪、磁力计、加速度计原始数据,然后离线用 Python 处理。
### 实验 1:静态漂移测试(最关键)
手机平放静止 30s 分钟。

陀螺仪:对 ωz 做累加积分得到 yaw(t),看曲线如何偏离 0,求漂移率(°/min)
磁力计:算 yaw(t) = atan2(By, Bx),求其标准差作为噪声水平

典型结果:陀螺仪是一条斜率不为零的近直线,磁力计是围绕真值抖动的横线。

### 实验 2:定角度旋转测试(绝对精度)
桌上用量角器画 0°/90°/180°/270° 参考线,手机依次旋转到每个位置静置几秒。

陀螺仪:积分得每段角度增量
磁力计:读每个静置点的航向角
算误差均值和重复性标准差

### 实验 3:回归原点测试(陀螺仪软肋)
绕 z 轴随意正反转 1 分钟,然后精确转回起始位置。

陀螺仪积分结果不会回到 0,残余角即累积误差
磁力计应直接回到初始读数,残余接近 0
